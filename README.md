# vesk-selfbot
Download Files as Zip.
First edit the "Config" Files and paste your token in the quotation marks.
Next run "runmefirst.bat" , this installs all the requirements.
After that, run "start.bat" if this has any errors, you need to delete all your python modules/packages and then re-run "runmefirst.bat".
